ティラノスクリプト テクニックサンプル集①
説明書　Ver 0.01

ティラノスクリプトのテクニックサンプルを集めたゲームです。
ゲームの中身は丁寧なコメント付きですべて見ることができます。

また、サンプルコードはそのままコピペして自分のプロジェクトで
使用することもできます、改変して使うことも問題ありません。

完全無償です。商用利用にも制限はありません。

■利用方法
index.htmlをティラノライダー（デバッグツール）で読み込んで
動作確認してください。

実際のスクリプトファイルはdata/scenario/ フォルダ以下に格納されています。

動作確認をして、参考にしたい箇所を見つけたら
その部分のスクリプトを確認してみるとよいでしょう。

■収録素材（画像や音楽）ファイルについて
掲載されている画像や音楽の素材ファイルを
ご自身のプロジェクトで使用することはできません。
（言及があるものは除く）
